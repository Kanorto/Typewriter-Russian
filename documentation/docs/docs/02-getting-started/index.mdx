# Первые шаги

Typewriter разработан так, чтобы быть гибким и простым в использовании. Он позволяет создавать интерактивный опыт для игроков через веб‑панель без написания кода. Это руководство поможет установить и настроить Typewriter, а на странице об интерфейсе объясняется работа веб‑панели.
