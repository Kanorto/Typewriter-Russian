# Начало работы

Typewriter создан с упором на гибкость и простоту.
Он позволяет создавать интерактивный контент для игроков через веб-панель без написания кода.
В этом разделе вы узнаете, как установить и настроить Typewriter, а также как пользоваться панелью.
