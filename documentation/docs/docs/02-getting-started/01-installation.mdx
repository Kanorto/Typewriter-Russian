# Руководство по установке

:::danger[Platform Compatibility]
Typewriter работает только на серверах **Paper**. На Spigot или Bukkit он функционировать не будет.
:::

## Установка плагина Typewriter

Чтобы корректно установить Typewriter на ваш Paper‑сервер Minecraft, выполните следующие шаги:

### Установка плагина

1. Сначала скачайте <DownloadVersion />.
2. Поместите загруженный файл плагина в папку `plugins` вашего сервера.

### Зависимости

Typewriter зависит от дополнительных плагинов, которые нужно установить для корректной работы.

3. Скачайте и добавьте зависимость Typewriter — [PacketEvents](https://modrinth.com/plugin/packetevents/versions?l=paper) — в каталог `plugins`.

4. Убедитесь, что у вас **нет** плагинов [InteractiveChat](https://www.spigotmc.org/resources/interactivechat-show-items-inventory-in-chat-custom-chat-keywords-bungee-velocity-support.75870/) или [CustomNamePlates](https://polymart.org/resource/customnameplates.2543) (версии 3.0.15 и ниже), так как они конфликтуют с Typewriter.

### Базовое расширение

Typewriter предлагает различные расширения для кастомизации, но обязательно установите `BasicExtension`.

5. Скачайте последнюю версию `BasicExtension` с <DownloadVersion /> и поместите её в папку `plugins/Typewriter/extensions`.

Полный список доступных расширений можно найти в [разделе расширений](../../adapters).

6. После добавления всех компонентов перезапустите сервер Minecraft, чтобы завершить установку Typewriter.

:::danger[Несоответствие версий]
При обновлении плагина обязательно **устанавливайте** соответствующие ему версии расширений. Несовпадение версий может привести к некорректной работе плагина!
:::

## Настройка веб‑панели

:::caution[Хостинг‑провайдеры]
Веб‑панель Typewriter **несовместима** с некоторыми внешними хостингами, такими как **Minehut**, **Aternos** и **Apex**. Обычно они не поддерживают открытие нескольких портов, хотя служба поддержки может открыть их по запросу.

Если открыть порты невозможно, вы всё равно сможете пользоваться остальными функциями Typewriter или настроить локальный сервер с установленным Typewriter.

Если нужна помощь, задавайте вопросы в нашем [Discord](https://discord.gg/HtbKyuDDBw).
:::

:::info[Потребление ресурсов]
Учтите, что веб‑панель и веб‑сокет потребляют ресурсы сервера, поэтому лучше размещать панель на сервере разработки, а не на рабочем.
:::

Теперь, когда плагин установлен, нужно настроить веб‑панель.

Для работы веб‑панели необходимо открыть два порта. Их можно изменить, но потребуется как минимум два свободных порта. По умолчанию используются `8080` и `9092`.

Чтобы включить веб‑панель, выполните следующие действия:

### Включение веб‑панели

1. Откройте файл `plugins/Typewriter/config.yml`.
2. Измените настройки следующим образом:

```yaml title="plugins/Typewriter/config.yml"
# Включены ли веб‑панель и веб‑сокеты.
enabled: true
# Имя хоста сервера. ИЗМЕНИТЕ на IP вашего сервера.
hostname: "127.0.0.1"
# Панель использует веб‑сокеты для синхронизации изменений с сервером и позволяет работать нескольким людям одновременно.
websocket:
  # Порт сервера веб‑сокета. Убедитесь, что он открыт.
  port: 9092
  # Используемая схема аутентификации. Не меняйте, если не знаете, что делаете.
  auth: "session"
panel:
  # Панель можно отключить, оставив сокеты открытыми. Делайте это только если понимаете последствия.
  # Если сокеты отключены, панель всегда будет выключена.
  enabled: true
  # Порт веб‑панели. Убедитесь, что он открыт.
  port: 8080
```

3. Проверьте с помощью [portchecker](https://portchecker.co/), что порты открыты. Если нет, откройте их самостоятельно или обратитесь к хостинг‑провайдеру.
4. Перезапустите сервер, чтобы завершить установку.

## Подключение к веб‑панели

Чтобы подключиться к веб‑панели, выполните следующие шаги:

1. Выполните команду `/typewriter connect` в игре. Вы получите ссылку для доступа к веб‑панели.

![Connect-command](../assets/installation/connect-command.gif)

2. Откройте полученную ссылку в вашем браузере.

<Image img={require("../assets/installation/connect-book.png")} width={400} alt="Connect to the web panel in the book" />

После подключения вы сможете использовать панель для создания и настройки квестов, диалогов NPC и многого другого. Через панель также можно просматривать и редактировать уже существующие взаимодействия игроков.
