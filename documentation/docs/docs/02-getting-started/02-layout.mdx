# Оформление панели

На этой странице вы узнаете, как пользоваться веб-панелью Typewriter и создавать разные элементы.

## Создание страницы

Чтобы создать страницу, нажмите `Add Page` и введите её название.
Оно не должно совпадать с уже существующими страницами.
Затем выберите один из четырёх типов:
* **Sequence**: последовательность действий, которые выполняются по порядку.
* **Static**: данные, не изменяющиеся в течение работы программы.
* **Cinematic**: набор действий с таймингом.
* **Manifest**: состояние взаимодействия, вычисляемое во время работы.

После создания страницы типа sequence рассмотрим, как устроены различные макеты.

## Базовый макет

<Image img={require("../assets/layout/layout.png")} alt="Sequence Page Layout" />

1. **Page list**: список созданных страниц, здесь же можно добавить новые.
2. **Action bar**: панель с несколькими кнопками:
   - **Staging Indicator**: оранжевый цвет показывает, что изменения ещё не активны на сервере,
     зелёный — что активны. Наведите курсор и нажмите «Publish», чтобы опубликовать изменения.
   - **Search bar**: поиск и создание записей.
   - **+ button**: добавление новых записей.

**Inspector**: область редактирования свойств выбранной записи.

3. **Entry Information**: информация о записи, включая ID и название.
4. **Fields**: параметры записи, доступные для редактирования.
5. **Operations**: действия, которые можно выполнить с записью.

## Макет sequence, static и manifest

- В центре отображаются все ваши записи.
- Колёсиком мыши можно приближать и отдалять страницу.
- Нажмите на запись, чтобы открыть её в инспекторе.
- Если удерживать запись несколько секунд, её можно перетащить и связать с другими.

## Макет cinematic

<Image img={require("../assets/layout/layout-cinematic.png")} alt="Cinematic Page Layout" />

1. **Segments**: внутри записи можно создавать сегменты, отображающиеся на дорожке. Сегменты не должны перекрываться.
2. **Track**: здесь видны все созданные сегменты, также можно изменить время их показа.
3. **Track Duration**: длительность дорожки в тиках Minecraft: 20 тиков равны одной секунде.
