# Интерфейс

На этой странице вы узнаете, как пользоваться веб‑панелью Typewriter и создавать различные элементы.

## Создание страницы

Чтобы создать страницу, нажмите кнопку `Add Page` и введите название. Оно не должно совпадать с уже существующим. Далее выберите один из четырёх типов:
* **Sequence** – последовательность взаимодействий, которые могут воспроизводиться одно за другим.
* **Static** – данные, которые не меняются во время работы программы.
* **Cinematic** – последовательность действий с таймингом.
* **Manifest** – состояние взаимодействия, вычисляемое во время выполнения.

После создания страницы последовательности рассмотрим, как работают различные макеты.

## Базовый макет

<Image img={require("../assets/layout/layout.png")} alt="Sequence Page Layout" />

1. **Список страниц** – здесь отображаются созданные страницы и есть возможность добавить новые.
2. **Панель действий** – набор кнопок слева направо:
   - **Индикатор публикации**: оранжевый означает, что изменения ещё не активны на сервере, зелёный – что они опубликованы. Наведите курсор, чтобы увидеть кнопку "Publish" и, при желании, опубликовать изменения.
   - **Строка поиска** – поиск и создание записей.
   - **Кнопка +** – создание новых записей.

**Инспектор** – панель редактирования выбранной записи.

3. **Информация о записи** – здесь отображается ID и название.
4. **Поля** – изменение свойств записи.
5. **Операции** – различные действия, которые можно совершать с записью.

## Макеты sequence, static и manifest

- В центре отображаются все ваши записи.
- Используйте колёсико мыши, чтобы приблизить или отдалить отображение.
- Нажмите на запись, чтобы открыть её в инспекторе.
- Если удерживать запись несколько секунд, её можно перетащить и соединить с другими.

## Макет cinematic

<Image img={require("../assets/layout/layout-cinematic.png")} alt="Cinematic Page Layout" />

1. **Сегменты** – внутри cinematic‑записи можно создавать сегменты, отображающиеся на дорожке. Сегменты не должны перекрываться.
2. **Дорожка** – здесь видны все созданные сегменты и их длительность.
3. **Длительность дорожки** – задаёт время воспроизведения. Время указывается в тиках Minecraft (20 тиков = 1 секунда).
