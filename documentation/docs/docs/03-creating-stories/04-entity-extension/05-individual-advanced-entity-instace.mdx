# Индивидуальный продвинутый экземпляр сущности

## Что такое `Individual Advanced Entity Instance`?

`Individual Advanced Entity Instance` — это тип Entity Instance, который позволяет **расчитывать активности отдельно для каждого игрока**.

В отличие от обычного `NPC Instance` (или более общего `Shared Entity Instance`), где одна активность используется всеми игроками, индивидуальный экземпляр создаёт **отдельную активность на каждого игрока**.

## Основные отличия от `NPC Instance`

| Характеристика       | `NPC Instance`             | `Individual Advanced Entity Instance` |
| -------------------- | -------------------------- | ------------------------------------- |
| Расчёт активности    | Одна общая активность      | Активность для каждого игрока         |
| Положение сущности   | Одинаковое для всех        | Зависит от конкретного игрока         |
| Когда использовать   | Общие сценарии             | Индивидуальное поведение              |

## Пример: патрулирующий NPC

Рассмотрим NPC с активностью **патруль**, настроенной через `Patrol Activity`.

* **`NPC Instance`:**
  Если NPC видят 200 игроков, вычисляется **лишь одна** активность патруля. Все игроки наблюдают NPC, идущего по **одному и тому же пути**.

* **`Individual Advanced Entity Instance`:**
  Здесь рассчитываются **200 активностей** патруля — **по одной на каждого игрока**. Путь NPC может **отличаться для разных игроков**, что требует **значительных ресурсов**.

---

## Когда использовать `Individual Advanced Entity Instance`

Используйте такой экземпляр **только если нужно поведение или позиция, отличающиеся для каждого игрока**, например, чтобы NPC следовал за игроком или показывал ему путь, оставаясь невидимым для остальных.

:::warning[Рекомендация]
Не применяйте этот тип экземпляра для обычных NPC или тех, кого видит много игроков. **По возможности используйте стандартный NPC Instance**, если только не требуется индивидуальное поведение.
:::
