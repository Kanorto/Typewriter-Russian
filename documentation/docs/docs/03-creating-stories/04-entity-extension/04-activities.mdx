# Entities Activities
:::info[Before starting]
First read [Entities & NPC's](./index.mdx) before starting this tutorial.
:::
Dans ce tutoriel, vous apprendrez à ajouter des activités à vos entités.

## What are activities?
En temps normal, les entités sont statiques. Heureusement, grâce aux activités, elles peuvent devenir dynamiques. Par exemple, vous pouvez créer une activité qui se déclenche lorsque le joueur s’approche de l’entité, ou encore à des moments précis, comme lorsqu’il lui parle. Vous pouvez même utiliser les Road Networks pour faire voyager les entités à travers différents nœuds !

## Adding an activity to an entity
Dans ce tutoriel, nous allons modifier Oliver afin qu’il se déplace automatiquement sur un Rode Network. Lorsqu’un joueur lui parlera, il s’arrêtera et se tournera vers lui.

Pour cela, nous allons utiliser une ``In Dialogue Activity`` qui nous permettra de définir une activité quand le joueur lui parle ainsi qu'une autre activité pour le reste du temps.
Nous allons donc modifier l'instance pour ajouter une ``In Dialogue Activity``.
<EntrySearch entryName="in_dialogue_activity"/>

## Configuring the activity
La ``In Dialogue Activity`` est une activité que active deux child activies en fonction de si le joueur lui parle ou noon.
<Image img={require("./assets/activities/in_dialogue_activity.png")} alt="In Dialogue Activty panel image" />

Pour le ``Talking Activity``, nous allons utiliser une ``Look Close Activity``. Cela signifie qu’Olivier va regarder le joueur quand il lui parlera.

Pour le ``Idle Activity``, nous allons utiliser une ``Patrol Activity``. Olivier va donc navoguer entre les Nodes d'un Road Network. Si vous voulez apprendre à en créer un, cela se passe sur cette page : [Road Network](../road-network). Il faudra ajouter les nodes que vous souhaitez utiliser en cliquant sur le <ActionButton button='capture'/>.
<Image img={require("./assets/activities/patrol_activity.png")} alt="Patrol Activty panel image" />

## Result
Lorsque l'on interagira avec Olivier, celui-ci nous regardera et arrètera de se déplacer. Quand on ne lui parle pas, il continuera de se déplacer entre les nodes du Road Network.
