# Взаимодействие с сущностью
:::info[Перед началом]
Рекомендуем сначала ознакомиться с разделом [Сущности и NPC](./index.mdx).
:::

В этом уроке вы узнаете, как взаимодействовать с сущностью.

## Добавление события взаимодействия
Сначала добавьте запись `Add On Entity Interact Event` в последовательность. Нажмите значок `+` в правом верхнем углу панели и найдите эту запись, затем добавьте её на страницу.

<EntrySearch entryName="entity_interact_event"/>

### Выбор определения
В инспекторе записи `New Entity Interact Event` в поле Definition выберите определение сущности, с которой хотите взаимодействовать (в нашем примере это Oliver).

<Image img={require("../../assets/entity-extension/entity-interact-event-fields.png")} alt="Entity Interact Event Fields" width={400} />

## Добавление события
Теперь добавим действие, которое будет выполняться при взаимодействии игрока с сущностью (в уроке это будет реплика). Щёлкните правой кнопкой по записи `New Entity Interact Event` и выберите `+ link with ...`, затем найдите `Add Spoken` и добавьте эту запись.

<EntrySearch entryName="spoken"/>

Установите в поле Speaker определение Oliver и введите текст `Hello I am Oliver.`.

## Результат
Опубликуйте страницы — в игре взаимодействие с Оливером будет выглядеть так:
