# Взаимодействие с сущностью
:::info[Перед началом]
Лучше сначала прочитать раздел [Сущности и NPC](./index.mdx) перед началом этого урока.
:::

В этом уроке вы узнаете, как взаимодействовать с сущностью.

## Добавление события взаимодействия
Сначала добавьте в последовательность запись `Add On Entity Interact Event`. Нажмите значок `+` в правом верхнем углу панели, найдите `Add On Entity Interact Event` и добавьте его на страницу.

<EntrySearch entryName="entity_interact_event"/>

### Выбор определения
В инспекторе `New Entity Interact Event` в поле definition выберите определение сущности, с которой хотите взаимодействовать (в нашем случае — Oliver)

<Image img={require("../../assets/entity-extension/entity-interact-event-fields.png")} alt="Entity Interact Event Fields" width={400} />

## Добавление события
Нужно добавить событие, которое будет срабатывать при взаимодействии игрока с сущностью. В этом уроке мы используем Spoken. Щёлкните правой кнопкой по записи `New Entity Interact Event` и выберите `+ link with ...`, затем найдите `Add Spoken` и добавьте его.

<EntrySearch entryName="spoken"/>

Установите спикером определение Oliver и задайте текст `Hello I am Oliver.`.

## Результат
Опубликуйте страницы, и в игре при взаимодействии с Оливером всё будет выглядеть так:
