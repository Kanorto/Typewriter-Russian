# Условное отображение NPC
:::warning[Создан NPC]
В этом уроке предполагается, что сущность уже создана.\
Если вы ещё этого не сделали, сначала пройдите [руководство по Entity Extension](./index.mdx).
:::
Здесь вы узнаете, как показывать сущность только при выполнении условий с помощью аудиторий.

## Добавление записи аудитории
Существует множество типов аудиторий. В этом уроке мы используем Game Time audience, но остальные работают похоже.


Сначала нажмите значок `+` в правом верхнем углу панели и найдите `New Game Time Audience`. Добавьте его на страницу.

<EntrySearch entryName="game_time_audience"/>

## Привязка к экземпляру сущности
Затем перетащите ваш экземпляр сущности на запись `New Game Time Audience` — так аудитория свяжется с сущностью.

Должно получиться примерно так:

<EntryDisplay pages={[
    require("./assets/conditional/manifest.json"),
]} />

## Настройка аудитории
В записи аудитории есть два поля:
* **Children** — здесь добавляются сущности, которые должны отображаться при выполнении условий.
* **Active Times** — показывает Children только когда условие выполняется.
* **Inverted** — инвертирует условие, то есть сущность будет показана, когда аудитория *не* выполнена.
