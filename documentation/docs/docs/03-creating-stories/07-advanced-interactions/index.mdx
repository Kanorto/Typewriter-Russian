# Продвинутые взаимодействия

В этом разделе рассматриваются продвинутые техники взаимодействий, которые расширяют фундаментальные концепции из руководства [Первые взаимодействия](../01-interactions/index.mdx).

Если вы создаёте сложные квесты, интерактивных NPC или уникальные головоломки, эти шаблоны помогут вывести ваши истории Typewriter на новый уровень.
