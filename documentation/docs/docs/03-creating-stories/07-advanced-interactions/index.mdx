# Продвинутые взаимодействия

Этот раздел посвящён более сложным техникам создания взаимодействий и опирается на базовые понятия из руководств [Первые взаимодействия](../01-interactions/index.mdx).

Независимо от того, создаёте ли вы запутанные квесты, интерактивных NPC или уникальные головоломки, эти приёмы помогут вывести ваши истории в Typewriter на новый уровень.
