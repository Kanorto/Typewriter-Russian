# Добавление диалогов

:::info[Перед началом]
Перед тем как приступать к этому разделу, рекомендуется прочитать главу [«Синематики»](./index.mdx).
:::
В этом руководстве вы узнаете, как добавлять диалоги в свои интеракции.

## Добавление диалога
В синематиках доступно несколько видов диалогов: `spoken`, `actionbar`, `title`, `subtitle`. В этом примере мы будем использовать **actionbar**, но остальные настраиваются схожим образом.

Сначала нажмите на значок + в правом верхнем углу панели и найдите `Add Actionbar Dialogue Cinematic`. Добавьте его на страницу кликом по +.
<EntrySearch entryName='actionbar_dialogue_cinematic'/>

### Настройка диалога
Теперь нужно настроить диалог.

Сначала добавьте [speаker](../01-interactions/index.mdx#configuring-the-speaker) для диалога. Затем перейдите в раздел операций и нажмите `Add Segment`, чтобы добавить сегмент.

### Настройка сегмента диалога
Внутри сегмента в поле Text можно добавить текст вроде `<white>Это крутой текст actionbar!</white>`.

<Image img={require("../../assets/cinematics/adding-dialogue-segment.png")} alt="Adding a dialogue segment" width={400} />

## Результат
Диалог успешно добавлен, и вы можете увидеть его в своём синематике.
