# Добавление диалогов

:::info[Before starting]
Перед началом этого руководства лучше прочитать раздел [Cinematics](./index.mdx).
:::
В этом уроке вы узнаете, как добавлять диалоги в свои интеракции.

## Добавление диалога
В кинематике доступно несколько типов диалогов: `spoken`, `actionbar`, `title`, `subtitle`. В этом руководстве мы используем `actionbar`, остальные работают аналогично.

Сначала нажмите значок + в правом верхнем углу панели и найдите `Add Actionbar Dialogue Cinematic`. Добавьте его на страницу, кликнув по значку +.
<EntrySearch entryName='actionbar_dialogue_cinematic'/>

### Настройка диалога
Теперь необходимо настроить диалог.

Сначала добавьте [спикера](../01-interactions/index.mdx#configuring-the-speaker) к диалогу. Затем в разделе операций нажмите `Add Segment`, чтобы создать сегмент.

### Настройка сегмента диалога
Внутри сегмента в поле Text можно указать текст, например `<white>This is a cool actionbar text!</white>`.

<Image img={require("../../assets/cinematics/adding-dialogue-segment.png")} alt="Adding a dialogue segment" width={400} />

## Результат
Теперь диалог успешно добавлен, и вы можете увидеть его в своей кинематике.
