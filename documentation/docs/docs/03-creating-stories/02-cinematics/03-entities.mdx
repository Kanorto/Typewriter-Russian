# Добавление сущностей

:::info[Before starting]
Перед началом этого руководства лучше прочитать раздел [Cinematics](./index.mdx).
:::
:::danger[Entity Extension]
Перед началом убедитесь, что установили расширение Entity Extension.
:::
В этом уроке вы узнаете, как добавлять сущностей в свои интеракции.

## Добавление сущности
Нажмите значок + в правом верхнем углу панели и найдите `Add Entity Cinematic`. Добавьте его на страницу, кликнув по значку +.

<EntrySearch entryName='entity_cinematic'/>
### Выбор определения
В инспекторе нажмите `Select a entity_definition` и выберите тип сущности, которую хотите использовать в качестве NPC. В этом руководстве выберем `Add Npc Definition`. Далее выберите страницу manifest или создайте её, нажав `Add manifest`.

<Player url={require("../../assets/cinematics/select-entity-definition.webm").default} />

Далее в этом NPC Definition следуйте руководству [entity-extension](../04-entity-extension/index.mdx), чтобы создать красивого NPC.

### Добавление сегментов
Нужно добавить сегмент в дорожку. Это делается через Operations в инспекторе, выберите `Add Segment`.

### Настройка сегмента
В сегменте необходимо добавить артефакт. Нажмите `Select a entity_cinematic_artifact` и выберите его или создайте новый через `Add Entity Cinematic Artifact`. Затем выберите или создайте статическую страницу.

### Запрос режима контента
Вернитесь на страницу кинематики и нажмите на значок <ActionButton button='capture'/> в инспекторе над артефактом. Откроется режим контента в Minecraft. Наводя курсор на предметы в инвентаре, можно узнать их назначение, но мы воспользуемся только книгой.
Нажав на книгу, Typewriter начнёт записывать ваши движения и предметы, которые были в руках при запуске режима.

#### Запись NPC
:::warning[Items in inventory]
Предметы, которые были в ваших руках до включения режима контента, окажутся в руках NPC.
:::
