# Добавление сущностей

:::info[Перед началом]
Перед началом рекомендуется ознакомиться с разделом [«Синематики»](./index.mdx).
:::
:::danger[Entity Extension]
Перед началом этого руководства необходимо установить расширение Entity Extension.
:::
В этом уроке вы узнаете, как добавлять сущности в свои интеракции.

## Добавление сущности
Нажмите на значок + в правом верхнем углу панели и найдите `Add Entity Cinematic`. Добавьте его на страницу кликом по +.

<EntrySearch entryName='entity_cinematic'/>
### Выбор определения
В инспекторе нажмите `Select a entity_definition` и выберите тип сущности, которую хотите использовать как NPC. В примере мы воспользуемся `Add Npc Definition`. Затем выберите страницу manifest или создайте новую через `Add manifest`.

<Player url={require("../../assets/cinematics/select-entity-definition.webm").default} />

После этого внутри созданного определения NPC можно воспользоваться руководством [Entity Extension](../04-entity-extension/index.mdx), чтобы создать красивого NPC.

### Добавление сегментов
Теперь нужно добавить сегмент в дорожку. Для этого в инспекторе в разделе Operations нажмите `Add Segment`.

### Настройка сегмента
В сегменте необходимо добавить артефакт. Нажмите `Select a entity_cinematic_artifact` и выберите существующий артефакт или создайте новый через `Add Entity Cinematic Artifact`. Затем выберите или создайте страницу static.

### Запрос режима контента
Вернитесь на страницу синематика и нажмите иконку <ActionButton button='capture'/> в инспекторе над артефактом. Откроется режим контента в Minecraft. Наведя курсор на предметы в инвентаре, можно узнать их назначение, но мы воспользуемся только книгой.
Нажав на книгу, вы запустите запись движений и предметов, которые держали в руках до перехода в режим контента.

#### Запись NPC
:::warning[Предметы в инвентаре]
Предметы, которые находились в руках до включения режима контента, будут находиться в руках NPC.
:::
