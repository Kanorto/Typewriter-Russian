# Кинематика для разных миров
:::info[Before starting]
Перед началом этого руководства лучше прочитать раздел [Cinematics](./index.mdx).
:::

В этом уроке вы научитесь создавать кинематику, привязанную к разным мирам. Это позволяет использовать один ролик на нескольких мирах.

## Создание динамической переменной
Сначала нужно создать динамическую переменную, которая будет определять мир игрока. Сделайте следующее:
1. Откройте вашу camera cinematic и выберите сегмент.
2. Внутри сегмента у одного из путей нажмите <ActionButton button='dynamic-variable'/>.
3. Найдите `Player World Position Variable` и добавьте её в сегмент.

<EntrySearch entryName='player_world_position_variable'/>

### Настройка динамической переменной
:::warning[repeat]
Повторите эти действия для всех путей внутри сегмента camera cinematic.
:::
В инспекторе **camera cinematic segment** укажите координаты вашей кинематики.
<Image img={require("../../assets/cinematics/world-specific-player-variable.png")} alt="Configuring the dynamic variable" width={400} />
#### Где значение мира?
Мир исчез из настроек, потому что мы используем `Player World Position Variable`.\
Используется мир, в котором находится игрок.
