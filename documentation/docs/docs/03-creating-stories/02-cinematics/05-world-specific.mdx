# Синематики для разных миров
:::info[Перед началом]
Перед началом рекомендуется прочитать раздел [«Синематики»](./index.mdx).
:::

В этом руководстве вы узнаете, как создавать синематики, зависящие от мира. Это позволяет использовать один синематик в нескольких мирах.

## Создание динамической переменной
Для начала нужно создать динамическую переменную, которая будет определять, в каком мире находится игрок. Выполните следующие шаги:
1. Откройте синематик с камерой и выберите сегмент
2. В сегменте у одного из путей нажмите иконку <ActionButton button='dynamic-variable'/>
3. Найдите `Player World Position Variable` и добавьте её в сегмент

<EntrySearch entryName='player_world_position_variable'/>

### Настройка динамической переменной
:::warning[repeat]
Повторите эти действия для всех путей в сегменте камеры.
:::
Теперь в инспекторе **сегмента камеры** можно задать координаты синематика.
<Image img={require("../../assets/cinematics/world-specific-player-variable.png")} alt="Configuring the dynamic variable" width={400} />
#### Куда пропало поле мира?
Хороший вопрос: значение мира убрано, потому что его определяет `Player World Position Variable`.\
Используется мир, в котором находится игрок.
