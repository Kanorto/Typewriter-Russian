# Пропускаемые кинематики
:::info[Before starting]
Перед началом этого руководства лучше прочитать раздел [Cinematics](./index.mdx).
:::
В этом уроке вы узнаете, как разрешить игрокам пропускать отдельные части кинематик.

## Добавление пропускаемой кинематики
Нажмите значок + в правом верхнем углу панели и найдите `Add Skippable Cinematic`. Добавьте её на страницу, кликнув по значку +.

<EntrySearch entryName='skip_cinematic'/>

### Настройка пропускаемой кинематики
В инспекторе поля confirmation key можно выбрать клавишу `SNEAK` или `SWAP_HANDS`. Игроку нужно нажать её, чтобы пропустить ролик. В этом примере используется SNEAK.

## Добавление пропускаемой части
В инспекторе пропускаемой кинематики нажмите `Add segment` в разделе operations, чтобы добавить новый сегмент.\
В полях Start и End frame задайте временной диапазон, в котором игрок может пропустить ролик. В примере это 0 и 100.

## Результат
Теперь у вас есть кинематика, которую можно пропустить, нажав выбранную клавишу.
