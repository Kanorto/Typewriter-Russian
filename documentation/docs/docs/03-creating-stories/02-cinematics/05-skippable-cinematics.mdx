# Пропускаемые синематики
:::info[Перед началом]
Перед началом рекомендуется ознакомиться с разделом [«Синематики»](./index.mdx).
:::
В этом руководстве вы узнаете, как дать игрокам возможность пропускать отдельные части синематиков.

## Добавление пропускаемого синематика
Нажмите на значок + в правом верхнем углу панели и найдите `Add Skippable Cinematic`. Добавьте его на страницу кликом по +.

<EntrySearch entryName='skip_cinematic'/>

### Настройка пропускаемого синематика
В инспекторе пропускаемого синематика в поле confirmation key можно выбрать `SNEAK` или `SWAP_HANDS`. Нажатие этой клавиши позволит игроку пропустить синематик. В примере используется клавиша SNEAK.

## Добавление пропускаемого участка
В инспекторе пропускаемого синематика нажмите `Add segment` в разделе operations, чтобы создать новый сегмент.\
В полях Start и End frame укажите интервал, в течение которого игрок может пропустить синематик. В примере это 0 и 100.

## Результат
Теперь у вас есть синематик, который можно пропустить, нажав выбранную клавишу.
