# Первое взаимодействие

:::info[Before starting]
Предполагается, что вы уже установили [Basic Extension](../../02-getting-started/01-installation.mdx#basic-extension).

Также убедитесь, что ознакомились с документом [Интерфейс](../../02-getting-started/02-layout.mdx) и создали страницу **sequence**.
:::

Создадим наше первое взаимодействие. В примере оно состоит из следующих шагов:

1. Игрок кликает по цветку.
2. Открывается диалог в чате.

Если у вас возникнут трудности, задавайте вопросы в нашем [Discord](https://discord.gg/HtbKyuDDBw).

## Что такое взаимодействия?

Взаимодействие происходит, когда игрок взаимодействует с чем‑то и тем самым запускает событие. В этом уроке кликом по цветку запускается диалог в чате.

## Создание взаимодействия

Нажмите на значок + в правом верхнем углу панели и найдите `on interact with block`. Добавьте эту запись на страницу, нажав на +.

<EntrySearch entryName='on_interact_with_block'/>

### Настройка взаимодействия

В инспекторе укажите блок `Red Tulip` (1) и имя `on flower click` (2).

<Image img={require("../../assets/interactions/on_flower_click.png")} alt="Configuring an interact with flow entry" width={400} />

## Создание события

Теперь создадим событие, которое будет запускаться при клике по цветку. Щёлкните правой кнопкой по записи `On Flower Click` и выберите `+ link with ...`. В появившемся меню найдите `Add Spoken` и добавьте его на страницу.

<Player url={require("../../assets/interactions/add-spoken.webm").default} />

### Настройка события

В инспекторе установите текст `Hello! I am a flower.` и длительность `1s`.

### Настройка говорящего

Выполните следующие действия:
1. Нажмите на текст `Select a speaker` и в меню выберите `Add Simple Speaker`.
2. Выберите `Add static` и задайте имя этой static‑странице.
3. Откройте созданный simple speaker.
4. В инспекторе установите Display Name `A beautiful flower`.

<Player url={require("../../assets/interactions/simple-speaker.webm").default} />
