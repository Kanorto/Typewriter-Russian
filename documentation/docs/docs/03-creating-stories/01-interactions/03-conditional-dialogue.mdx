# Условный диалог
:::info[Перед началом]
Перед началом лучше прочитать [First Interaction](./index.mdx) и [Facts](../03-facts/index.mdx).
Также убедитесь, что у вас создана страница **sequence**.
:::

В этом руководстве вы научитесь создавать условный диалог. Если у игрока определённый факт имеет нужное значение, будет показан соответствующий текст.

## Добавление spoken
Сначала добавьте spoken. Как это сделать, описано в уроке [First Interaction](./index.mdx).

## Настройка spoken
В поле текста spoken укажем `Hey you clicked me for the first time.`

### Настройка критериев
Нужно задать критерии для spoken. Нажмите значок + в инспекторе рядом с полем criteria и выберите нужный факт. В нашем примере это постоянный факт `Flower Clicked` с группой игрока.

<Image img={require("../../assets/interactions/flower-clicked-fact.png")} alt="Flower Clicked Fact" width={400}/>

Внутри criteria задайте оператор `==` и значение `0`.

### Настройка модификатора
После настройки критериев нужно изменить факт так, чтобы при повторном нажатии на цветок появился другой spoken. Нажмите значок + возле поля modifier и выберите факт, который хотите изменить. В примере используем тот же факт. В модификаторе установите оператор `=` и значение `1`.

<Image img={require("../../assets/interactions/add-spoken-fields.png")} alt="Add Spoken Fields" width={400}/>

## Добавление второго spoken
Теперь нужно добавить второе spoken. Щёлкните правой кнопкой по записи `On Interact Block Event` и выберите `+ Link with ...`. Найдите `Add Spoken` и добавьте его в последовательность.\
В текстовом поле укажите `Wow you clicked me again!`
Страница последовательности должна выглядеть так:

<Image img={require("../../assets/interactions/conditional-dialogue-sequence.png")} alt="Conditional dialogue sequence" width={400}/>

### Настройка критериев второго spoken
Задайте критерии для второго spoken: нажмите значок + рядом с полем criteria, снова выберите факт `Flower Clicked`. Внутри criteria установите оператор `==` и значение `1`.

### Дополнительный диалог
Чтобы добавить ещё диалоги или действия при нажатии на цветок, создайте ещё один модификатор с оператором `=` и значением `2` (или любым другим) и повторите шаги.

## Результат
При первом нажатии на цветок появится первый spoken, при повторном — второй.
