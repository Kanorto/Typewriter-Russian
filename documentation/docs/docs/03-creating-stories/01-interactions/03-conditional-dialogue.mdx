# Условный диалог
:::info[Before starting]
Перед началом стоит прочитать [Первое взаимодействие](./index.mdx) и [Факты](../03-facts/index.mdx).
Также убедитесь, что у вас создана страница **sequence**.
:::

В этом руководстве вы узнаете, как создавать условные диалоги. Если у игрока определённый факт имеет заданное значение, будет показан соответствующий диалог.

## Добавление spoken
Сначала нужно добавить spoken. Сделать это можно по шагам из руководства [Первое взаимодействие](./index.mdx).

## Настройка spoken
В поле text spoken укажем `Hey you clicked me for the first time.`

### Настройка критериев
Теперь настроим критерии для spoken. Нажмите на значок + в инспекторе рядом с полем criteria и выберите нужный факт. В примере мы используем постоянный факт `Flower Clicked` с группой игрока.

<Image img={require("../../assets/interactions/flower-clicked-fact.png")} alt="Flower Clicked Fact" width={400}/>

Внутри критериев установите оператор `==` и значение `0`.

### Настройка модификатора
Критерии заданы, теперь нужно изменить факт, чтобы при повторном клике выводился другой spoken. Нажмите + рядом с полем modifier и выберите тот же факт. В модификаторе установите оператор `=` и значение `1`.

<Image img={require("../../assets/interactions/add-spoken-fields.png")} alt="Add Spoken Fields" width={400}/>

## Добавление второго spoken
Теперь добавим второй spoken. Щёлкните правой кнопкой по записи `On Interact Block Event` и выберите `+ Link with ...`. Найдите `Add Spoken` и добавьте его в последовательность.\
В поле text укажем `Wow you clicked me again!`
Ваша страница последовательности будет выглядеть так:

<Image img={require("../../assets/interactions/conditional-dialogue-sequence.png")} alt="Conditional dialogue sequence" width={400}/>

### Критерии для второго spoken
Настроим критерии второго spoken. Нажмите + рядом с полем criteria и снова выберите факт `Flower Clicked`. Внутри критериев установите оператор `==` и значение `1`.

### Дополнительный диалог
Чтобы добавить больше диалогов или действий при клике на цветок, задайте ещё один модификатор с оператором `=` и значением `2` (или любым другим) и повторите предыдущие шаги.

## Результат
При первом клике на цветок будет показан первый spoken. При повторном клике — второй.
