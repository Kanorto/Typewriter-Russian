# Добавление вариантов

:::info[Перед началом]
Перед началом рекомендуется прочитать руководство [First Interaction](./index.mdx).
:::

В этом уроке вы узнаете, как создавать и использовать варианты в интеракциях.

## Создание варианта

Чтобы создать вариант, выполните следующие шаги:

1. Щёлкните правой кнопкой по записи и выберите `+ Link with ...`.
2. Найдите `Add Option` и нажмите на значок +, чтобы добавить его в последовательность.

<EntrySearch entryName='option' />

## Настройка варианта

Настроим вариант, выполнив следующие действия:

1. В поле `Text` введите вопрос, который хотите задать игроку, например `Am i a beautiful flower?`.
2. Выберите подходящего [говорящего](./index.mdx#configuring-the-speaker) для варианта.

<Image img={require("../../assets/interactions/configure-option.png")} alt="Configuring an option entry" />

### Добавление вариантов ответа

1. Нажмите на значок + рядом с полем `Options` и разверните `Options #1`.
2. В поле `Text` для первого варианта введите ответ, например `Yes you are!`.
3. Нажмите значок + в области triggers внутри `Options #1` и разверните `Triggers #1`.
4. Нажмите `Select a triggerable` и в меню поиска создайте запись, которая выполнится после выбора варианта. Например, spoken с текстом `Yay thank you!`.
5. Повторите эти шаги, чтобы создать другие варианты, например `No you are not.`.

<Image img={require("../../assets/interactions/adding-options.png")} alt="Adding options to an option entry" />
