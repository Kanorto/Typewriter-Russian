# Добавление вариантов

:::info[Before starting]
Перед началом лучше прочитать [Первое взаимодействие](./index.mdx).
:::

В этом руководстве вы узнаете, как создавать и использовать варианты во взаимодействиях.

## Создание варианта

Чтобы добавить вариант, выполните следующие действия:

1. Щёлкните правой кнопкой по записи и выберите `+ Link with ...`.
2. Введите `Add Option` и нажмите на значок +, чтобы добавить запись в последовательность.

<EntrySearch entryName='option' />

## Настройка варианта

Настроим вариант следующим образом:

1. В поле `Text` введите вопрос, который хотите задать игроку, например `Am i a beautiful flower?`.
2. Выберите нужного [говорящего](./index.mdx#configuring-the-speaker) для этого варианта.

<Image img={require("../../assets/interactions/configure-option.png")} alt="Configuring an option entry" />

### Добавление ответов

1. Нажмите на значок + возле поля `Options` и разверните `Options #1`.
2. В поле `Text` впишите ответ для первого варианта, например `Yes you are!`.
3. Нажмите + в поле triggers внутри `Options #1` и раскройте `Triggers #1`.
4. Нажмите `Select a triggerable` и в открывшемся меню создайте запись, которая будет запускаться при выборе этого варианта. Например, spoken с текстом `Yay thank you!`.
5. Повторите эти шаги, чтобы добавить больше вариантов, например `No you are not.`.

<Image img={require("../../assets/interactions/adding-options.png")} alt="Adding options to an option entry" />
