# Выдача предметов

:::info[Перед началом]
Перед началом желательно ознакомиться с уроком [First Interaction](./index.mdx).
:::

В этом руководстве вы узнаете, как выдавать предметы в интеракциях.

## Добавление выдачи предмета

Чтобы добавить элемент выдачи предмета, выполните следующие действия:

1. Щёлкните правой кнопкой по записи и выберите `+ Link with ...`.
2. Найдите `Add give item` и нажмите значок +, чтобы добавить его в последовательность.

<EntrySearch entryName='give_item' />

## Настройка выдачи предмета

Запись выдачи предмета имеет два типа: пользовательский предмет и сериализованный предмет.

### Различия
<Tabs groupId="Differences" queryString>
  <TabItem value="custom-item" label="Custom Item" default>
    Пользовательский предмет позволяет настроить почти все его параметры.
    :::warning[NBT]
    Такой предмет не может содержать данные NBT.
    :::
    </TabItem>
    <TabItem value="serialized-item" label="Serialized Item" default>
    Сериализованный предмет даёт возможность использовать Item Capturer и захватывать предмет из игры.
    :::warning[Not Editable]
    Редактировать сериализованный предмет нельзя, можно изменить только имя.
    :::
    </TabItem>
</Tabs>

### Использование Item Capturer (сериализованный предмет)

Проще всего воспользоваться захватом предмета: он копирует предмет, который у вас в руке в игре, прямо в панель. Для этого нажмите значок <ActionButton button='capture'/> рядом с полем `item`.

<Player url={require("../../assets/interactions/item-capturer.webm").default} />

### Настройка вручную (пользовательский предмет)

Предмет можно настроить и самостоятельно. В поле item выберите `Custom Item`. Нажмите значок + справа от текста `Components(0)`, чтобы добавить один из следующих компонентов:
1. **Amount**: количество выдаваемых предметов.
2. **Flag**: особые параметры предмета.
3. **Lore**: описание предмета.
4. **Material**: собственно предмет, который получит игрок.
5. **Item Name**: имя предмета.
