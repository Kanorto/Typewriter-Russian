# Выдача предметов

:::info[Before starting]
Перед началом лучше прочитать [Первое взаимодействие](./index.mdx).
:::

В этом руководстве вы узнаете, как выдавать предметы во взаимодействиях.

## Добавление выдачи предмета

Чтобы добавить запись выдачи предмета, выполните следующие шаги:

1. Щёлкните правой кнопкой по записи и выберите `+ Link with ...`.
2. Найдите `Add give item` и нажмите на значок +, чтобы добавить её в последовательность.

<EntrySearch entryName='give_item' />

## Настройка выдачи предмета

Запись выдачи предмета имеет два режима: пользовательский предмет и сериализованный предмет.

### Отличия
<Tabs groupId="Differences" queryString>
  <TabItem value="custom-item" label="Custom Item" default>
    Пользовательский предмет позволяет настроить почти все его параметры.
    :::warning[NBT]
    Пользовательский предмет не может содержать NBT‑данные.
    :::
    </TabItem>
    <TabItem value="serialized-item" label="Serialized Item" default>
    Сериализованный предмет позволяет использовать Item Capturer и скопировать предмет прямо из игры.
    :::warning[Not Editable]
    Отредактировать сериализованный предмет нельзя, можно изменить только имя.
    :::
    </TabItem>
</Tabs>

### Использование Item Capturer (Serialized Item)

Проще всего воспользоваться захватом предмета, который просто копирует предмет в вашей руке в панель. Для этого нажмите на значок <ActionButton button='capture'/> рядом с полем `item`.

<Player url={require("../../assets/interactions/item-capturer.webm").default} />

### Настройка вручную (Custom Item)

Предмет можно настроить и вручную. В поле item выберите `Custom Item`. Нажмите значок + рядом с текстом `Components(0)`, чтобы добавить один из следующих компонентов:
1. **Amount**: количество выдаваемых предметов.
2. **Flag**: дополнительные опции предмета.
3. **Lore**: описание предмета.
4. **Material**: предмет, который получит игрок.
5. **Item Name**: имя предмета.
