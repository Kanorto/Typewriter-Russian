# Случайный диалог
:::info[Before starting]
Это руководство основано на уроке [первого взаимодействия](./index.mdx). Также рекомендуется ознакомиться с документацией по [Фактам](../03-facts/index.mdx).
:::

Здесь вы узнаете, как использовать динамические переменные и создавать случайный spoken с их помощью.

## Создание случайного диалога
Откройте инспектор spoken из урока [первого взаимодействия](./index.mdx). Затем нажмите <ActionButton button='dynamic-variable'/> над полем Text и выберите `Add random Variable`.

<EntrySearch entryName='random_variable' />

Это создаст случайную переменную внутри static‑страницы.

## Настройка случайной переменной
После создания переменной откройте её инспектор. Здесь можно добавить несколько значений. Каждый раз при срабатывании spoken будет выводиться случайное сообщение из этой переменной.\
В примере мы добавили строки `Hello! I am a flower.`, `Wow, you found me.`, `Hey you!`, но вы можете указать любые.\
Теперь ваша переменная будет выглядеть так:
<Image img={require("../../assets/interactions/random-variable.png")} alt="Random Variable Example" width={400}/>

## Удаление случайной переменной
Чтобы удалить динамическую переменную, нажмите на значок <ActionButton button='remove'/> рядом с ней.

## Результат
Теперь каждый раз, когда spoken будет срабатывать, из переменной будет выводиться случайное сообщение.
