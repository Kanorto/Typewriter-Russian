# Случайный диалог
:::info[Перед началом]
Этот урок основан на [first interaction](./index.mdx). Также желательно сначала прочитать раздел [Facts](../03-facts/index.mdx).
:::

В этом уроке вы узнаете, как использовать динамические переменные и создавать случайные spoken с их помощью.

## Создание случайного диалога
Сначала откройте инспектор spoken, созданного в уроке [first interaction](./index.mdx). Затем нажмите <ActionButton button='dynamic-variable'/> над полем Text и выберите `Add random Variable`.

<EntrySearch entryName='random_variable' />

Это создаст случайную переменную внутри статической страницы.

## Настройка случайной переменной
Откройте инспектор созданной переменной и добавьте несколько значений. Теперь каждый раз при срабатывании spoken будет показываться одно из этих сообщений.\
В примере мы добавили строки `Hello! I am a flower.`, `Wow, you found me.`, `Hey you!`, но вы можете использовать любые свои варианты.\
Ваша динамическая переменная должна выглядеть так:
<Image img={require("../../assets/interactions/random-variable.png")} alt="Random Variable Example" width={400}/>

## Удаление случайной переменной
Чтобы удалить динамическую переменную, нажмите значок <ActionButton button='remove'/> рядом с ней.

## Результат
Теперь при каждом срабатывании spoken будет показываться случайное сообщение из переменной.
