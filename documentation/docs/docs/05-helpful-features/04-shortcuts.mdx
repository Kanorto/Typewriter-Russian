# Горячие клавиши
На этой странице приведён список сочетаний клавиш, которые можно использовать в Typewriter.\
Но сначала выберите вашу операционную систему.
<Tabs groupId="operating-systems">
  <TabItem value="windows" label="Windows">
| Сочетание                                                  | Описание                                                                              |
| ----------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| `ctrl + k `or` ctrl + space`                              | Открыть окно поиска                                                                          |
| `tab `or` ctrl + n`                                       | Следующий элемент                                                                                |
| `shift + tab  `or` ctrl +  p`                             | Предыдущий элемент                                                                            |
| `ctrl + shift + p`                                        | Опубликовать страницы                                                                            |
  </TabItem>
  <TabItem value="mac" label="macOS">
| Сочетание                                                  | Описание                                                                              |
| ----------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| `⌘ + k `or` ⌘ + space`                                  | Открыть окно поиска                                                                          |
| `tab `or` ⌘ + n`                                         | Следующий элемент                                                                                |
| `shift + tab  `or` ⌘ +  p`                               | Предыдущий элемент                                                                            |
| `⌘ + shift + p`                                          | Опубликовать страницы                                                                            |
  </TabItem>
</Tabs>

## Поиск
В меню поиска также есть несколько сочетаний.

| Сочетание                                                  | Описание                                                                              |
| ----------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| `!ae` or `!ne`                                            | Показывать только **новые** записи |
| `!ee` or `!a`                                             | Показывать только **существующие** записи |

Нажимая Tab в меню поиска, можно быстро просматривать все записи. Внизу также отображаются действия вроде `+Add` или `Open wiki`.
