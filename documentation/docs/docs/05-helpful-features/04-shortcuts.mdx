# Горячие клавиши
На этой странице перечислены сочетания клавиш, которые можно использовать в Typewriter.\
Сначала выберите вашу операционную систему.
<Tabs groupId="operating-systems">
  <TabItem value="windows" label="Windows">
| Сочетание | Описание |
| --------- | -------- |
| `ctrl + k` или `ctrl + space` | Открыть строку поиска |
| `tab` или `ctrl + n` | Следующий пункт |
| `shift + tab` или `ctrl + p` | Предыдущий пункт |
| `ctrl + shift + p` | Опубликовать страницы |
  </TabItem>
  <TabItem value="mac" label="macOS">
| Сочетание | Описание |
| --------- | -------- |
| `⌘ + k` или `⌘ + space` | Открыть строку поиска |
| `tab` или `⌘ + n` | Следующий пункт |
| `shift + tab` или `⌘ + p` | Предыдущий пункт |
| `⌘ + shift + p` | Опубликовать страницы |
  </TabItem>
</Tabs>

## Поиск
В меню поиска тоже есть полезные сочетания.

| Сочетание | Описание |
| --------- | -------- |
| `!ae` или `!ne` | Показывать только **новые** записи |
| `!ee` или `!a` | Показывать только **существующие** записи |

Нажимая Tab, можно быстро листать записи. Внизу появится меню с действиями `+Add` или `Open wiki`.
