# Главы

Главы упрощают работу и напоминают структуру книги. Как файлы в папках, страницы группируются по главам.

### Создание главы
Создайте главу: щёлкните по странице правой кнопкой, выберите `Change Chapter`, введите название и нажмите `Change`.
<Player url={require("../assets/chapters/chapters.webm").default} />

### Создание подглав
Создавать подглавы просто: добавьте к имени главы точку. Например, `hello.test` создаст подглаву. Ниже пример:

<Image img={require("../assets/chapters/sub-chapters.png")} alt="Subchapters" width={400} />


Можно добавить сколько угодно подглав, например `hello.test.morning`. Typewriter группирует страницы с тем же именем.
