# Главы

Главы добавляют удобство в работу с Typewriter, напоминая привычную структуру глав в книге. Подобно тому как файлы образуют папки на компьютере, система глав в Typewriter группирует страницы внутри глав.

### Создание глав
Создайте главу, щёлкнув правой кнопкой мыши по странице и выбрав `Change Chapter`. Введите нужное название и нажмите `Change`.
<Player url={require("../assets/chapters/chapters.webm").default} />

### Создание подглав
Создавать подглавы очень просто. Используйте основное название главы и добавьте `.`. Например, `hello.test` создаст подглаву, как показано ниже:

<Image img={require("../assets/chapters/sub-chapters.png")} alt="Subchapters" width={400} />


При желании расширяйте историю сколь угодно подглавами. Например, `hello.test.morning` тоже подойдёт. Typewriter автоматически собирает страницы с одинаковым названием главы и группирует их вместе.
