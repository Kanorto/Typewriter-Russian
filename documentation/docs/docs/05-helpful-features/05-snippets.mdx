# Что такое сниппеты?

Сниппеты — небольшие блоки информации, которые используют расширения для отображения.
Они позволяют настраивать разные части адаптеров, например отображение сообщений в `Basic Extension`.

# Как настроить сниппеты?
Сниппеты можно изменить, редактируя файл `plugin/Typewriter/snippets.yml`.

:::info[important]
Учтите: сниппеты создаются при первом использовании. Чтобы изменить, сначала запустите запись, которая их использует.
:::

После первого получения `MessageDialogueEntry` файл `snippets.yml` будет таким:

```yaml title="plugin/Typewriter/snippets.yml"
dialogue:
  message:
    format: |2

      <gray> | <bold><speaker></bold><reset><gray> |
      <reset><white> <message>
```

После этого отредактируйте файл. Например, замените `<bold><speaker></bold>` на `<bold><red><speaker></red></bold>`.

Yaml использует символ `|` для обозначения многострочной строки.

```yaml title="plugin/Typewriter/snippets.yml"
dialogue:
  message:
    // highlight-next-line
    format: |2

      <gray> | <bold><speaker></bold><reset><gray> |
      <reset><white> <message>
```
Строка `|2` означает многострочный текст с двумя пустыми строками в конце. Если они не нужны, оставьте `|`.
