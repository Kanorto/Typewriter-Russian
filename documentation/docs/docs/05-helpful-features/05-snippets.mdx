# Что такое Snippet'ы?

Snippet'ы — это небольшие фрагменты информации, которые могут использовать расширения. В основном они относятся к тому, как что-либо отображается пользователям.
Snippet'ы позволяют настраивать различные части адаптеров. Например, можно изменить способ отображения сообщений в `Basic Extension`.

# Как настроить Snippet'ы?
Настройку можно выполнить, отредактировав файл `plugin/Typewriter/snippets.yml`.

:::info[important]
Обратите внимание, что Snippet'ы записываются только при первом использовании. Поэтому, если вы хотите изменить snippet, сначала вызовите запись, использующую его.
:::

Для `Basic Extension` после того как первый игрок получит `MessageDialogueEntry`, файл `snippets.yml` будет выглядеть так:

```yaml title="plugin/Typewriter/snippets.yml"
dialogue:
  message:
    format: |2

      <gray> | <bold><speaker></bold><reset><gray> |
      <reset><white> <message>
```

После этого вы можете отредактировать файл, чтобы изменить snippet. Например, чтобы поменять цвет имени говорящего по умолчанию, замените `<bold><speaker></bold>` на `<bold><red><speaker></red></bold>`.

В YAML символ `|` обозначает многострочную строку.

```yaml title="plugin/Typewriter/snippets.yml"
dialogue:
  message:
    // highlight-next-line
    format: |2

      <gray> | <bold><speaker></bold><reset><gray> |
      <reset><white> <message>
```
`|2` указывает, что строка многострочная и содержит `2` перевода строки в конце. Если лишние строки не нужны, уберите `2`, оставив просто `|`.
