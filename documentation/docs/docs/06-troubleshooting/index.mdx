# Руководство по устранению неполадок

Добро пожаловать в руководство по устранению неполадок плагина Typewriter. Здесь собраны решения распространённых проблем при использовании плагина для Paper Spigot. Прежде чем обращаться за помощью в Discord или на GitHub, ознакомьтесь с этим списком типичных ошибок и способов их устранения. Такой подход поможет быстрее решить вашу проблему.

## Панель не загружается

Если панель не загружается, выполните следующие действия:

1. Убедитесь, что сервер запущен и доступен.
2. Проверьте наличие сетевых проблем.
3. Убедитесь, что настройки брандмауэра не блокируют доступ.

Подробнее об устранении этой проблемы см. раздел [Панель не загружается](troubleshooting/ports).

## Ошибка: «plugin.yml не найден» при установке расширения

Если при установке расширения вы видите ошибку, похожую на эту:

```log
[08:49:35 ERROR]: [DirectoryProviderSource] Error loading plugin: Directory 'plugins/BasicExtension.jar' failed to load!
java.lang.RuntimeException: Directory 'plugins/BasicExtension.jar' failed to load!
        ...
Caused by: java.lang.IllegalArgumentException: Directory 'plugins/BasicExtension.jar' does not contain a paper-plugin.yml or plugin.yml! Could not determine plugin type, cannot load a plugin from it!
```

Обычно это значит, что расширение размещено не в той папке. Смотрите [Правильное размещение расширений](./extensions.mdx#correct-extension-placement) для инструкций.

## Плейсхолдеры не распознаются

Если плейсхолдеры из PlaceholderAPI, например `%player_name%`, не распознаются в Typewriter, выполните следующие шаги:

1. Найдите нужное дополнение PlaceholderAPI. Список доступных дополнений см. [здесь](https://github.com/PlaceholderAPI/PlaceholderAPI/wiki/Placeholders).
2. Установите его командой `/papi ecloud download [extension_name]`. Для `%player_name%` выполните `/papi ecloud download player`.
3. Проверьте работу плейсхолдера командой `/papi parse me <placeholder>`. Для `%player_name%` используйте `/papi parse me %player_name%`.
4. Если плейсхолдер работает в обычном режиме, но не в Typewriter, попробуйте перезапустить сервер.

Если проблема не устраняется, создайте тикет в нашем [Discord](https://discord.gg/HtbKyuDDBw) для получения дополнительной помощи.
