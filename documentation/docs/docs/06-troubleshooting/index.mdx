# Руководство по устранению неполадок

Добро пожаловать в руководство по поиску и устранению проблем Typewriter.
Здесь собраны советы по решению распространённых ошибок при использовании плагина с Paper Spigot.
Прежде чем обращаться за помощью в Discord или на GitHub, ознакомьтесь с этим списком.
Возможно, ответ уже есть и вы быстро найдёте решение.

## Панель не загружается

Если панель не загружается, выполните следующие действия:

1. Убедитесь, что сервер запущен и доступен.
2. Проверьте наличие сетевых проблем.
3. Удостоверьтесь, что настройки брандмауэра не блокируют подключение.

Более подробные шаги описаны здесь: [панель не загружается](troubleshooting/ports).

## Ошибка «plugin.yml not found» при установке расширения

Если при установке расширения появляется ошибка, похожая на эту:

```log
[08:49:35 ERROR]: [DirectoryProviderSource] Error loading plugin: Directory 'plugins/BasicExtension.jar' failed to load!
java.lang.RuntimeException: Directory 'plugins/BasicExtension.jar' failed to load!
        ...
Caused by: java.lang.IllegalArgumentException: Directory 'plugins/BasicExtension.jar' does not contain a paper-plugin.yml or plugin.yml! Could not determine plugin type, cannot load a plugin from it!
```

Обычно это означает, что расширение размещено неверно.
Правильное расположение описано здесь: [где хранить расширения](./extensions.mdx#правильное-расположение-расширений).

## Плейсхолдеры не обрабатываются

Если плейсхолдеры из PlaceholderAPI, например `%player_name%`, не подставляются в Typewriter, выполните следующие шаги:

1. Найдите нужное расширение PlaceholderAPI.
   Список доступен [здесь](https://github.com/PlaceholderAPI/PlaceholderAPI/wiki/Placeholders).
2. Установите его командой `/papi ecloud download [название]`. Для `%player_name%` это `/papi ecloud download player`.
3. Проверьте работу плейсхолдера командой `/papi parse me <placeholder>`.
   Для нашего примера: `/papi parse me %player_name%`.
4. Если плейсхолдер работает в обычном режиме, но не в Typewriter, перезапустите сервер.

Если проблема не решена, создайте тикет на нашем [Discord](https://discord.gg/HtbKyuDDBw) и мы поможем вам.
