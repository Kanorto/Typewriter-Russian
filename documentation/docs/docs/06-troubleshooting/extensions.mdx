# Устранение неполадок расширений

Возникли проблемы с расширениями Typewriter?
В этом руководстве вы найдёте поэтапные инструкции по их правильной настройке и работе вместе с плагином Typewriter.

## Правильное расположение расширений

Учтите, что расширения Typewriter — это не обычные Paper‑плагины.
Это специальные JAR‑файлы только для Typewriter, поэтому их нужно хранить отдельно.
Для корректной работы выполните следующее:

1. Перейдите в каталог плагина: `/plugins/Typewriter/extensions`.
2. Если папки `extensions` нет, создайте её для хранения расширений.
3. Поместите все файлы расширений в эту папку.

Отдельная папка позволяет Typewriter читать и использовать расширения, не смешивая их с обычными плагинами Paper.

## Совместимость версий

Чтобы плагин Typewriter и его расширения работали без конфликтов,
они должны иметь одинаковую версию. Это важно как при обновлении, так и при откате:

1. Обновляя `typewriter.jar`, одновременно обновляйте все расширения до той же версии.
2. При понижении версии действуйте аналогично — плагин и расширения должны совпадать.

Соблюдение единой версии предотвращает проблемы совместимости и обеспечивает стабильную работу.
