В этом руководстве вы узнаете, как заставить Typewriter работать с playit.gg.

## Что такое Playit.gg
Playit.gg — сервис, который позволяет сделать ваш локальный Minecraft‑сервер общедоступным. Это удобно для тестирования сервера с друзьями или для краткосрочного публичного запуска.

## Как заставить Typewriter работать с playit.gg
Сначала перейдите на [playit.gg](https://playit.gg/), создайте агента и подключите свой Minecraft‑сервер. В интернете есть множество руководств, как это сделать.

### Создание нового туннеля
Во вкладке Tunnels нажмите `+ Add tunnel`.

<Image img={require("../assets/troubleshooting/tunnels.png")} alt="Tunnels" />

В появившемся окне выберите регион и тип туннеля `TCP (protocol)`, затем укажите локальный порт `8080` (или порт, указанный в config.yml) и нажмите `Add tunnel`.

<Image img={require("../assets/troubleshooting/create-tunnel.png")} alt="Create tunnel" />

Туннель сначала будет ожидать подключения, а через несколько секунд должен заработать.

### Подключение к панели Typewriter
Теперь вы можете подключиться к панели, используя URL, указанный во вкладке Tunnels.

<Image img={require("../assets/troubleshooting/ip.png")} alt="Tunnel url" />
