## Что такое Playit.gg
Playit.gg — это сервис, который позволяет сделать локальный сервер Minecraft общедоступным.
Это удобно для тестирования с друзьями или непродолжительного публичного запуска сервера.

## Как настроить Typewriter с playit.gg
Сначала перейдите на [playit.gg](https://playit.gg/), создайте агента и подключите к нему свой сервер Minecraft.
В сети есть много подробных инструкций по этому процессу.

### Создание нового туннеля
На вкладке Tunnels нажмите `+ Add tunnel`.

<Image img={require("../assets/troubleshooting/tunnels.png")} alt="Tunnels" />

В появившемся окне выберите регион и тип `TCP (protocol)`.
Затем укажите локальный порт `8080` (или тот, что задан в `config.yml`) и нажмите `Add tunnel`.

<Image img={require("../assets/troubleshooting/create-tunnel.png")} alt="Create tunnel" />

Созданный туннель некоторое время будет иметь статус Pending, после чего станет активным.

### Подключение к панели TW
Теперь можно открыть панель по адресу, указанному на вкладке Tunnels.
