# Проблемы с загрузкой панели

Если панель плагина Typewriter не загружается, это руководство поможет найти и устранить причину.

## Корректное назначение портов

Часто панель не открывается из‑за отсутствия свободных портов. Typewriter требует два открытых порта:

1. Один для загрузки панели.
2. Второй для веб‑сокета, через который сервер обменивается данными.

:::warning
Хостинги вроде Aternos, предоставляющие лишь один порт, не подходят для работы с Typewriter.
:::

## Правильный адрес

Проверьте, что для доступа к панели используется верный адрес.
Его формат должен быть `ip:port`, а при использовании поддомена — `url:port`.

## Порт в конфигурации

Частая ошибка — указание порта в поле `hostname` файла `config.yml`.
Там должен быть только IP‑адрес или доменное имя, без порта. Например:

```yaml title="config.yml"
# ...
// highlight-red
hostname: "127.0.0.1:25565" // Incorrect line
// highlight-green
hostname: "127.0.0.1"       // Correct line
# ...
```

Удаление порта из `hostname` может решить проблему с загрузкой панели.
