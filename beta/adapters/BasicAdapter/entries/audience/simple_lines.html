Simple Lines

The SimpleSidebarLinesEntry is a display that shows lines.

Separating lines with a newline character will display them on separate lines.

How could this be used?​

This could be used to show a list of objectives, or the region a player is in.

Fields​

Lines

RequiredColoredMulti-LinePlaceholders

The lines to display on the sidebar. Separate lines with a newline character.


Colors and formatting from the MiniMessage Adventure Api can be used. So for example, you can use <red>Some Text</red> for red text.

Placeholders from the PlaceholderApi can be used. So for example, you can use %player_name% for the player name.
